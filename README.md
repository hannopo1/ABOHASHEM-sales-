# ABOHASHEM-sales- — منصة الذكاء المالي والتشغيلي المتكاملة

منصة تحليل مالي وتشغيلي متكاملة لشركة **أبوهاشم للحوم — Food Industries**، مبنية بالكامل من الملفات المرفوعة في هذا المستودع (فواتير مبيعات، تصنيف أصناف بالعلامة التجارية، سعة كرتونة/تعبئة، أرصدة مديونية 8 مناديب مبيعات). لا توجد أي بيانات مُفترَضة أو مُستقاة من مصدر خارجي — كل رقم في هذا المستودع قابل للتتبع حتى الملف المصدري الذي استُخرج منه.

## الملفات المصدر (كما رُفعت، دون تعديل)

| الملف | المحتوى |

## بنية المستودع

```
analysis/     نصوص Python لخط أنابيب التحليل الكامل (تشغيلها بالترتيب الرقمي 01→10)
data/processed/  كل مخرجات التحليل (CSV/JSON) — قابلة لإعادة التوليد بالكامل من الملفات المصدر
dashboards/   لوحة تحكم تفاعلية واحدة (index.html) بثماني لوحات فرعية + تبعياتها
reports/      3 تقارير Markdown: جودة البيانات، التقرير الأكاديمي الاقتصادي القياسي، التقرير الاستثماري التنفيذي
```

## إعادة توليد التحليل بالكامل

```bash
pip install -r analysis/requirements.txt
cd /path/to/repo
for f in analysis/0*.py analysis/1*.py; do python3 "$f"; done
```

كل نص برمجي مستقل، يقرأ من الملفات المصدر و/أو مخرجات النص السابق له في `data/processed/`، ويكتب مخرجاته الخاصة هناك. الخطوة الأخيرة (`10_export_dashboard_data.py`) تُجمّع كل شيء في `dashboards/data.js` الذي تقرأه لوحة التحكم.

## لوحة التحكم التفاعلية

افتح `dashboards/index.html` مباشرة في أي متصفح (لا يتطلب خادمًا أو اتصال إنترنت — مكتبة الرسوم البيانية Chart.js مُضمَّنة محليًا في `dashboards/vendor/`). تحتوي على 8 تبويبات:

1. **الملخص المالي التنفيذي** — إيراد، اتجاه + تنبؤ، جسر الإيرادات، مديونية حسب المندوب، تركّز المخاطر
2. **المبيعات التنفيذي** — أعلى/أدنى أداء، باريتو 80/20، Treemap، الشجرة الهرمية القابلة للتوسّع (عميل ← علامة ← صنف)
3. **ربحية العملاء** — تصنيف ABC/XYZ، مصفوفة التقاطع، تركّز الإيراد، جدول تفصيلي لأعلى 25 عميلاً
4. **أداء العلامات التجارية** — اتجاه شهري، حصص، متوسط سعر البيع، توقع 7 أشهر لكل علامة
5. **أداء المنتجات** — مصفوفة ABC×XYZ (فقاعية)، اتجاه الإيراد والسعر لأهم 10 أصناف
6. **التصنيع والتعبئة** — احتياج تعبئة ضمني مُشتق من الكميات المباعة + سعة الكرتونة، مؤشر الموسمية
7. **التنبؤ (7 أشهر)** — مقارنة نماذج، 3 سيناريوهات + فترة ثقة 95%، توقعات مُفصّلة للعملاء/الأصناف
8. **جودة البيانات** — مطابقة الاستخراج، القيم المفقودة، الاختبارات الإحصائية القياسية

## التقارير

- `reports/data_quality_report.md` — تقرير جودة بيانات كامل (مطابقة، قيم مفقودة، تسوية أسماء، شذوذ)
- `reports/academic_econometric_report.md` — تقرير اقتصادي قياسي أكاديمي (ADF/KPSS/Ljung-Box/Chow/… + تفسير خماسي الطبقات + ربط بالأدبيات + قيود)
- `reports/investor_executive_report.md` — تقرير استثماري تنفيذي لمجلس الإدارة والجهات التمويلية، مع توصيات مرتّبة حسب الأثر التجاري

## قيد جوهري صريح

**لا تحتوي الملفات المرفوعة على أي بيانات تكلفة** (لا COGS، لا تكلفة إنتاج، لا مصروفات تشغيلية). لذلك **لا يظهر في أي تقرير أو لوحة هامش ربح إجمالي/تشغيلي أو EBITDA أو صافي ربح** — عرض هذه المؤشرات كان سيتطلب افتراض تكلفة غير موجودة في المصدر، وهو ما يخالف صراحة قيد "عدم اختلاق بيانات". كل التحليل المالي هنا مبني على الإيراد الصافي المرصود فعليًا.
